操作系统复习

chapter6 设备管理

设备管理

设备分类

按速度分：

  1. 低速设备
  2. 中速设备
  3. 高速设备

按单位分：

  1. 块设备
  2. 字符设备

按资源分配角度：

  1. 独占设备
  2. 共享设备
  3. 虚拟设备

设备管理任务目标

  1. 实现设备并行性从而提高系统利用率
  2. 设备动态分配，从而提高设备利用率
  3. 采用多种缓冲，从而缩小cpu和设备速度差异
  4. 屏蔽设备物理特性

I/O方式

  1. 直接控制方式
  2. 中断方式
  3. DMA方式
    以数据块为单位
  4. 通道
    以数据块为单位
    是独立于cpu的专用处理机

缓冲技术

目的：

  1. 改善cpu与外围设备之间速度不匹配的矛盾
  2. 减少I/O对cpu中断次数
  3. 提高cpu和I/O设备并行性

常用的缓冲区技术有：单缓冲、双缓冲、多缓冲（循环缓冲、缓冲池）

磁盘

  1. 物理结构
    磁盘是一种直接存取设备、又叫随机存取设备

  2. 磁盘访问时间

    1. 寻道时间
    2. 旋转延迟时间
    3. 传输时间

磁盘调度算法

  1. 先来先服务
  2. 最短寻道时间优先
  3. 扫描算法
    消除了饥饿现象
  4. 循环扫描算法

设备分配

数据结构：

  1. 设备控制表（DCT）
  2. 系统设备表（SDT）
  3. 控制器控制表（COCT）
  4. 通道控制表（CHCT）

虚拟设备

SPOOLing系统

  1. 组成部分

    • 输入井、输出井
      外存大缓冲区
    • 输入缓冲区、输出缓冲区
      内存
    • 输入进程、输出进程

  2. 原理
    把一个共享的硬盘改造成若干台输入、输出设备。这样的设备称为虚拟设备

  3. 特点

    • 提高了I/O速度
    • 设备不被任何进程独占
    • 实现了虚拟设备
